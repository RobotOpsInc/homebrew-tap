# Robot Ops Homebrew Tap

Homebrew formulae for [Robot Ops](https://robotops.com) open source tools.

## Install

```sh
brew install robotopsinc/tap/rosql
```

## Documentation

- [rosql.org](https://rosql.org) — quickstart, CLI reference, examples
- [RobotOpsInc/rosql](https://github.com/RobotOpsInc/rosql) — source, issues, releases

## Issues

Formula bugs → open an issue here. CLI bugs → [RobotOpsInc/rosql](https://github.com/RobotOpsInc/rosql/issues).
